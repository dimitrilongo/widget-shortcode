=== Widget Shortcode ===
Contributors: shazdeh
Plugin Name: Widget Shortcode
Tags: widget, shortcode, theme, admin
Requires at least: 3.0
Tested up to: 3.3
Stable tag: 0.1

Adds a [widget] shortcode which enables you to output a widget anywhere you like.

== Description ==

The shortcode requires the widget ID, but no need to guess, the plugin generates the code for you. You can also use before_widget, after_widget, before_title and after_title parameters to override the widget arguments. here's an example: <code>[widget id="text-1" before_title="<h3>" after_title="</h3>"]</code>

== Installation ==

1. Upload the whole plugin directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's it. use the [widget] shortcode anywhere you want.
4. Enjoy!
