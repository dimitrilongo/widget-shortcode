Give you 4 sidebars that show up before and after your content on single pages, and above and below your blog posts.

== Description ==

The before and after content areas show up only in single views, and above and below loop show up, you guessed it, above and below your posts. This plugin will work with any standard theme. Also checkout my <a href="http://wordpress.org/extend/plugins/widgets-in-columns/">Widgets in Columns</a> plugin which enables you to divide the widgets in any widget area into your desired columns.


== Installation ==

1. Upload the whole plugin directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Goto Appearance -> Widgets
4. Enjoy!
